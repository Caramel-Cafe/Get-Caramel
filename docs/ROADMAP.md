# Roadmap

## Phase 1 - Foundation
- Monorepo layout
- Domain boundaries
- Environments and CI skeleton
- Design system baseline

## Phase 2 - Auth + Roles
- Phone/email auth
- JWT + refresh token flow
- Roles: customer, vendor_owner, courier, admin

## Phase 3 - Vendor + Catalog
- Vendor profile + KYC hooks
- Menu categories, items, options, add-ons
- Availability windows

## Phase 4 - Customer Ordering
- Home feed + search + filters
- Store page + item detail + cart
- Address and checkout

## Phase 5 - Fulfillment
- Order state machine
- Kitchen acceptance flow
- Courier dispatch and reassignment

## Phase 6 - Realtime
- Live courier location
- ETA updates
- Push notifications

## Phase 7 - Payments
- Card/wallet/cash support
- Vendor settlement + payout schedule
- Basic fraud checks

## Phase 8 - Quality + Ops
- Reviews and reporting
- Admin moderation and support tools
- SLA dashboards

## Phase 9 - Scale + Security
- Load tests
- RBAC auditing
- Backups and incident runbooks
